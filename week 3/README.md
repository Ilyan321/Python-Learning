1. Contact Management System (Mega Project 1 & 5)
A full-featured, menu-driven application to manage personal contacts using Python dictionaries.

2. Robust Division Calculator (Mega Project 2)
A program designed to perform division while gracefully handling potential runtime crashes .


Error Handling: Implements try-except blocks to catch ZeroDivisionError and ValueError (for non-numeric inputs) .


Logic: Uses float conversion to ensure precision in division results .

3. Data Structure Operations (Mega Project 3 & 4)
A deep dive into Python's built-in data structures to perform statistical and organizational tasks.


Tuple Operations: A function that accepts a tuple and returns the maximum, minimum, and total sum simultaneously .


List Manipulation: Demonstrates the use of .sort() and .reverse() methods to organize numerical data in descending order .

4. Simple Banking System
A simulation of a banking environment focusing on global state management.

Features: Check balance, deposit funds, and withdraw funds with real-time balance updates.

Concepts: Utilization of the global keyword to manage the amount variable across multiple functional scopes.

5. Advanced Menu-Driven Calculator (Assignment Day 5)
An interactive calculator that persists until the user chooses to exit.


Operational Checks: Uses membership testing (not in) to validate if a user-selected operator is supported .


Nested Handling: Features nested try-except blocks specifically for division operations within a broader calculation loop .
