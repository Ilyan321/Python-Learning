3. Mega Project 3: Number Guessing Game
An interactive game where the user attempts to guess a randomly generated number between 1 and 10.


File: Mega_Project_3.py 

Features:

Round Management: Play multiple rounds with 3 attempts per round.


Randomization: Uses the random library for game logic.
